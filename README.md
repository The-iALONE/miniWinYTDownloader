# winYouTube Downloader V0.0.1

This project is a mini YouTube Downloader for Windows OS under development.


## Instalation

Extract Release.rar file then you see two files:

```https
  miniWinYouTubeDownloader Setup.msi
  setup.exe <-- Run This
```

Run setup.exe and install it.
After installation go to the path you specified for installation and run miniWinYTDownloader.exe
Done!
